CSV PARSER DOCUMENTATION

DESCRIPTION

this app service is used to parse and convert CSV file to JSON array.

USAGE

  1. /api/v1/upload
  2. methods - post
  3. payload - csv file
  4. enctype - multipart/form-data
  5. mimetype - text/csv or text/comma-separated-values
  6. fieldname - csv

USAGE

send a post request to /api/v1/upload, the fieldname of the uploaded file must be csv and the column headers must be name, organization, award, date, description, certificate_number

EXAMPLE USAGE

visit /api/v1/upload/test, upload a csv file with the following heading:

  1. name
  2. organization
  3. description
  4. award
  5. date
  6. certificate_number

view returned JSON array
